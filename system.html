Navigation

Desktop: persistent left sidebar (260px) with primary navigation.
Mobile (< 960px): hidden sidebar, hamburger reveals a slide-in sheet. A 5-item bottom nav stays anchored at the screen base.

Search

Global search appears in the top bar (desktop), on the home hero, and via a bottom-nav button (mobile). Pressing ⌘K / Ctrl+K from anywhere opens the modal.

Governance fields (every page)

Owner · Last updated · Next review · Status (Active / Draft / Needs Review / Archived). Built into the resource-card meta and dept-card "Governance" panel.
